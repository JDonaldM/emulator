SWIFT Emulator
==============

Emulator for SWIFT run outputs, used to predict outputs of simulations
without having to run them. Employs Gaussian Process Regression with
`george` and sensitivity analysis with `SALib`.

Authors:

+ Roi Kugel
+ Josh Borrow


Requirements
------------

+ `george`
+ `SALib`
+ `attrs`
+ `numpy`
+ `unyt`
